    Section 2.5 Generalized eigenvectors

    Up to this point, eigenvalues and eigenvectors have helped us find some standard forms of operators. In particular, we have seen that if an operator \(T\) on \(V\text{,}\) an \(n\)-dimensional vector space has \(n\) linearly independent eigenvectors, then \(T\) is diagonalizable. We also know that this condition holds if \(T\) has \(n\) distinct eigenvalues or if the operator is self-adjoint.
    However, there are examples where this does not apply. For instance, the matrix \(A=\begin{bmatrix} 2 \amp 1 \\ 0 \amp 2 \\ \end{bmatrix}\) has a single eigenvalue \(\lambda=2\) and the associated eigenspace \(E_2\) is one-dimensional. In this case, the characteristic polynomial is \((\lambda-2)^2\) so the eigenvalue \(\lambda=2\) is a root with multiplicity two.
    This example show that just looking at eigenvalues and eigenvectors will not be enough to always find a standard form. So instead, we do something that mathematicians love to do: generalize an idea that has already proven useful. In this case, the eigenvalue/eigenvector condition is given by the equation
    \begin{equation*} (T-\lambda I)\vvec = \zerovec\text{.} \end{equation*}
    We will generalize this equation to
    \begin{equation*} (T-\lambda I)^k\vvec = \zerovec \end{equation*}
    for some \(k\) and call the solutions generalized eigenvectors.

    Subsection 2.5.1 Generalized eigenvectors

    Definition 2.5.1.

    If \(T\) is an operator on \(V\text{,}\) we say that a nonzero vector \(\vvec\) is a generalized eigenvector if \((T-\lambda I)^k\vvec=\zerovec\) for some \(k\text{.}\) The set of such vectors, the generalized eigenspace is denoted \(G_\lambda\text{.}\)
    Notice that eigenvectors are also generalized eigenvectors.
    First notice that \(G_k\) is an invariant subspace of \(V\) because \(G_k=\nul(p(T))\) for a polynomial \(p(x)=(x-\lambda)^k\text{.}\) Moreover, the minimal polynomial of \(T|_{G_k}\) is \(p(x)=(x-\lambda)^k\text{.}\) Since the minimal polynomial of \(T|_{G_k}\) divides the minimal polynomial of \(T\text{,}\) we know that \((x-\lambda)^k\) divides minimal polynomial of \(T\text{.}\) This means that \(\lambda\) is an eigenvalue of \(T\text{.}\) So while we have generalized the notion of eigenvector, the values of \(\lambda\) for which we have generalized eigenvectors and still eigenvalues of \(T\text{.}\)

    Proposition 2.5.2.

    A nonzero vector cannot belong to generalized eigenspaces corresponding to two different eigenvalues.

    Proof.

    Suppose that \(\lambda\) and \(\mu\) are eigenvalues and that \((T-\lambda I)^k\vvec=\zerovec\) for some \(k\text{.}\) Suppose also that \((T-\mu I)^j\vvec=\zerovec\) for some other \(m\text{.}\) Then
    \begin{equation*} \begin{aligned} \zerovec=(T-\mu I)^m\vvec \amp = (T-\lambda I + (\lambda-\mu)I)^m\vvec \\ \amp = \sum_{j=0}^m{m \choose j}(\lambda-\mu)^{m-j}(T-\lambda I)^j\vvec \\ \end{aligned}\text{.} \end{equation*}
    For this vector \(\vvec\text{,}\) there is a smallest value of \(k\) such that \((T-\lambda I)^k\vvec=\zerovec\text{,}\) but \((T-\lambda I)^{k-1}\vvec\neq\zerovec\text{.}\) We therefore multiply both sides of the previous expression by \((T-\lambda I)^{k-1}\) so that
    \begin{equation*} (T-\lambda I)^{k-1}\zerovec = \sum_{j=0}^m(\lambda-\mu)^{m-j} (T-\lambda I)^{k-1+j}\vvec = (\lambda - \mu)^m(T-\lambda I)^{k-1}\vvec\text{,} \end{equation*}
    which says that \(\lambda - \mu = 0\text{.}\)
    Moreover, a set of nonzero vectors \(\vvec_j\in G_{\lambda_j}\) for a different set of eigenvalues is linearly independent.

    Proposition 2.5.3.

    A set of generalized eigenvectors, each associated to a different eigenvalue, is linearly independent.

    Proof.

    Suppose that \(\vvec_1,\vvec_2,\ldots,\vvec_m\) is a set of generalized eigenvectors associated to different eigenvalues. We will proceed using induction on the number of vectors \(m\) in the set. If \(m=1\text{,}\) we know that
    \begin{equation*} a_1\vvec_1 = \zerovec \end{equation*}
    implies that \(a_1=0\text{.}\)
    Now suppose that \(m\gt 1\) and that the result is true for all smaller values of \(m\text{.}\) Suppose that \((T-\lambda_jI)^{k_j}\vvec_j=\zerovec\) and that
    \begin{equation*} a_1\vvec_1+a_2\vvec_2 + \ldots + a_m\vvec_m = \zerovec\text{.} \end{equation*}
    Multiply by \((T-\lambda_m)^{k_m}\) so that the last term on the left size becomes zero. Then we have
    \begin{equation*} a_1(T-\lambda_m)^{k_m}\vvec_1+ \ldots + a_m(T-\lambda_m)^{k_m}\vvec_{m-1} = \zerovec\text{.} \end{equation*}
    Since a generalized eigenvector can only be associated to one eigenvalue, each vector \((T-\lambda_m)^{k_m}\vvec_j\) is nonzero. In addition, each of these vectors is still a generalized eigenvector because
    \begin{equation*} (T-\lambda_j)^{k_j}(T-\lambda_m)^{k_m)\vvec_j = (T-\lambda_m)^{k_m)(T-\lambda_j)^{k_j}\vvec_j = \zerovec\text{.} \end{equation*}
    Therefore we have a linear combination of \(m-1\) generalized eigenvectors with
    \begin{equation*} a_1(T-\lambda_m)^{k_m}\vvec_1+ \ldots + a_m(T-\lambda_m)^{k_m}\vvec_{m-1} = \zerovec\text{.} \end{equation*}
    By induction \(a_1=a_2=\ldots=a_{m-1} = 0\text{,}\) which also implies that \(a_m=0\text{.}\) The original set of \(m\) generalized eigenvectors is then seen to be linearly independent.

    Subsection 2.5.2 Complex Vector Spaces

    Because of the Fundamental Theorem of Algebra, operators on complex vector spaces have special properties. In the rest of this section, we will focus on complex vector spaces.

    Proposition 2.5.4.

    If \(T\) is an operator on \(V\text{,}\) a complex vector space, then there is a basis of \(V\) consisting of generalized eigenvectors of \(T\text{.}\)

    Proof.

    Once again we proceed by induction on \(n\text{,}\) the dimension of \(V\text{.}\) The result is certainly true if \(n=1\) because every nonzero vector is an eigenvector of \(T\text{.}\)
    Now assume that \(n\gt 1 \) and that the result holds for all smaller values of \(n\text{.}\) Suppose that \(\lambda\) is an eigenvalue of \(T\) and consider the operator \((T-\lambda I)^n\text{.}\) We will first explain why
    \begin{equation*} V = \nul(T-\lambda I)^n \oplus \range(T-\lambda I)^n\text{.} \end{equation*}
    First, we can see that these subspaces of \(V\) intersect only in the zero vector. Suppose that \(\vvec\in\range(T-\lambda I)^n\text{,}\) which means there is a vector \(\uvec\) such that \((T-\lambda I)^n\uvec = \vvec\text{.}\) If \(\vvec\) is also in \(\nul(T-\lambda I)^n\text{,}\) then
    \begin{equation*} (T-\lambda I)^n\vvec = (T-\lambda I)^{2n}\uvec = \zerovec\text{.} \end{equation*}
    Because \(\dim V = n\text{,}\) remember that \(\nul(T-\lambda)^n =\nul(T-\lambda I)^{n+k}\) for any positive \(k\text{.}\) Since \(\uvec\in\nul(T-\lambda I)^{2n}\text{,}\) we also have \(\uvec\in\nul(T-\lambda I)^n\text{,}\) which means that
    \begin{equation*} \vvec = (T-\lambda I)^n\uvec = \zerovec\text{.} \end{equation*}
    Therefore, these two subspaces only intersect in the zero vector.
    In the usual way, the dimensions of these subspaces add to \(\dim V =n\) so we have
    \begin{equation*} V = \nul(T-\lambda I)^n \oplus \range(T-\lambda I)^n\text{.} \end{equation*}
    If \(\nul(T-\lambda I)^n = V\text{,}\) then every vector in \(V\) is a generalized eigenvector of \(T\) and the conclusion of the theorem holds. Suppose instead that
    \begin{equation*} \{\zerovec\} \subsetneq \nul(T-\lambda I)^n \subsetneq V \end{equation*}
    where the first inclusion holds because \(\lambda\) is an eigenvalue of \(T\text{.}\) This means that
    \begin{equation*} 0 \lt \dim \range(T-\lambda I)^n \lt n\text{.} \end{equation*}
    Now \(\range(T-\lambda I)^n\) is invariant under \(T\text{.}\) By induction, there is a basis of \(\range(T-\lambda I)^n\) consisting of generalized eigenvectors of \(T\text{.}\) Putting this together with a basis of \(\nul(T-\lambda I)^n\) gives a basis of \(V\) consisting of generalized eigenvectors of \(T\text{.}\)

    Proposition 2.5.5.

    If \(T\) is an operator on the complex vector space \(V\) and the set of eigenvalues of \(T\) is \(\lambda_1,\lambda_2,\ldots,\lambda_m\text{,}\) then
    \begin{equation*} V = G_{\lambda_1}\oplus G_{\lambda_2}\oplus\ldots \oplus G_{\lambda_m}\text{.} \end{equation*}

    Proof.

    We have seen that there is a basis of \(V\) consisting of generalized eigenvectors. Therefore, every vector in \(V\) can be written as a linear combination of generalized eigenvectors.
    We have further seen a set of generalized eigenvectors associated to different eigenvalues is linear independent.
    These two facts are enough to establish the theorem.

    Subsection 2.5.3 Jordan form

    We are now ready to proof the main structure theorem, again assuming that \(V\) is a complex vector space. By a Jordan block, we mean a matrix \(J\) whose diagonal entries are all \(\lambda\text{,}\) whose entries directly above the diagonal are 1, and whose other entries are all zero. That is,
    \begin{equation*} J = \begin{bmatrix} \lambda \amp 1 \amp 0 \amp \ldots \amp 0 \\ 0 \amp \lambda \amp 1 \amp \ldots \amp 0 \\ \vdots \amp \vdots \amp \ddots \amp \ldots \amp \vdots \\ 0 \amp 0 \amp 0 \amp \ldots \amp \lambda \\ \end{bmatrix}\text{.} \end{equation*}

    Theorem 2.5.6. Jordan canonical form.

    If \(T\) is an operator on a complex vector space \(V\text{,}\) then there is a basis for \(V\) such that the matrix associated to \(T\) consists only of Jordan blocks on the diagonal. That is,
    \begin{equation*} \begin{bmatrix} J_1 \amp 0 \amp \ldots \amp 0 \\ 0 \amp J_2 \amp \ldots \amp 0 \\ \vdots \amp \vdots \amp \ddots \amp 0 \\ 0 \amp 0 \amp \ldots \amp J_m \\ \end{bmatrix}\text{.} \end{equation*}

    Proof.

    We know that
    \begin{equation*} V = G_{\lambda_1}\oplus G_{\lambda_2}\oplus\ldots \oplus G_{\lambda_m}\text{.} \end{equation*}
    Moreover, on \(G_{\lambda_j}\text{,}\) the operator \((T-\lambda_j I)^{k_j}\) is nilpotent, which means there is a basis for \(G_{\lambda_j}\) such that the matrix associated to \(T-\lambda_j\) conists of nilpotent blocks. The matrix associated to \(T\) therefore has consists of Jordan blocks. Because each generalized eigenspace is invariant under \(T\text{,}\) the theorem holds.
    Notice that the characteristic polynomial of \(T\text{,}\) which can easily be found using this matrix, has the form
    \begin{equation*} p(x)=(x-\lambda_1)^{m_1}(x-\lambda_2)^{m_2}\ldots(x-\lambda_k)^{m_k} \end{equation*}
    where the multiplicity of each eigenvalue equals the dimension \(\dim G_{\lambda_j}\text{.}\) We earlier called \(m_j\) the algebraic multiplicity of the eigenvalue \(\lambda_j\text{.}\) Because the eigenspace \(E_{\lambda_j}\subset G_{\lambda_j}\text{,}\) have therefore shown that the multiplicity of the eigenvalue is at least as large as the dimension of the eigenspace:
    \begin{equation*} 0\lt \dim E^{\lambda_j} \leq m_j\text{.} \end{equation*}
