    Section 2.3 Orthogonaliaty

    As we’ve seen in our earlier studies, the dot product and orthogonality lead to many simplifications. For instance, the inverse of a matrix whose columns form an orthonormal basis of \(\real^n\) is just the transpose of that matrix. Since we are now looking to find bases that represent a given linear transformation with a simple form, it makes sense to start to consider orthogonal bases.

    Subsection 2.3.1 The adjoint of an operator

    We suppose now that \(V\) is a vector space over either \(\field=\real\) or \(\complex\) having an inner product denoted by \(\inner{\cdot}{\cdot}\text{.}\) If \(T\) is an operator on \(V\text{,}\) we can define its adjoint \(T^*\) through the following relationship
    \begin{equation*} \inner{T\vvec}{\wvec}=\inner{\vvec}{T^*\wvec} \end{equation*}
    for every \(\vvec\) and \(\wvec\) in \(V\text{.}\) Notice that this also means that
    \begin{equation*} \inner{T^*\wvec}{\vvec} = \inner{\wvec}{T\vvec}\text{.} \end{equation*}
    The first thing to establish is that \(T^*\) is itself an operator.

    Proposition 2.3.1.

    The adjoint \(T^*:V\to V\) is an operator on \(V\text{.}\)

    Proof.

    We first need to establish that \(T^*\wvec\) is a vector in \(V\) for every \(\wvec\) in \(V\text{.}\)
    Beginning with an orthonormal basis \(\uvec_1,\uvec_2,\ldots,\uvec_n\) for \(V\text{,}\) we have
    \begin{equation*} \uvec = \inner{\uvec}{\uvec_1}\uvec_1 + \ldots + \inner{\uvec}{\uvec_n}\uvec_n \end{equation*}
    for all vectors \(\uvec\text{.}\) We then define
    \begin{equation*} T^*\wvec = \inner{T^*\wvec}{\uvec_1}\uvec_1 + \ldots + \inner{T^*\wvec}{\uvec_n}\uvec_n = \inner{\wvec}{T\uvec_1}\uvec_1 + \ldots + \inner{\wvec}{T\uvec_n}\uvec_n\text{,} \end{equation*}
    which gives a definition of the vector \(T^*\wvec\text{.}\)
    Finally, we need to show that \(T^*\) satisfies the two linearity properties. Suppose that \(\wvec_1\) and \(\wvec_2\) are vectors in \(V\text{.}\) Then
    \begin{equation*} \inner{\vvec}{T^*(\wvec_1+\wvec_2)} = \inner{T\vvec}{\wvec_1+\wvec_2} = \inner{T\vvec}{\wvec_1} + \inner{T\vvec}{\wvec_2} =\inner{\vvec}{T^*\wvec_1 + T^*\wvec_2}\text{.} \end{equation*}
    Since this holds for any vector \(\vvec\text{,}\) we have
    \begin{equation*} T^*(\wvec_1+\wvec_2) = T^*\wvec_1 + T^*\wvec_2\text{.} \end{equation*}
    In the same way, we see that \(T^*(s\wvec) = sT^*\wvec\text{,}\) which verifies that \(T^*\) is an operator on \(T\text{.}\)
    We now investigate the matrices associated to \(T\) and \(T^*\) with respect to an orthonormal basis of \(V\text{.}\) As before, we use \(\uvec_1,\ldots,\uvec_n\) to denote an orthonormal basis of \(V\text{.}\)

    Proposition 2.3.2.

    If \(M\) is the matrix associated to \(T\text{,}\) then \(M^*\text{,}\) the matrix associated to \(T^*\) is
    \begin{equation*} A^* = \conjugate{A}^T\text{,} \end{equation*}
    the conjugate transpose of \(A\text{.}\)

    Proof.

    Remember how the matrix \(A\) is defined:
    \begin{equation*} T\uvec_j = \sum A_{i,j}\uvec_i\text{,} \end{equation*}
    which says that
    \begin{equation*} A_{i,j}=\inner{T\uvec_j}{\uvec_i}=\inner{\uvec_j}{T^*\uvec_i} = \conjugate{inner{T^*\uvec_i}{\uvec_j}} = \conjugate{B_{j,i}} \end{equation*}
    if \(B\) is the matrix defined by \(T^*\text{.}\)

    Subsection 2.3.2 Self-adjoint operators

    We that \(T\) is a self-adjoint operator if \(T^*=T\text{.}\) Notice that if \(V\) is a real vector space, this means that the matrix associated to \(T\) with respect to an orthonormal basis is symmetric. Here is a useful fact about self-adjoint operators.

    Lemma 2.3.3.

    If \(T\) is a self-adjoint operator on \(V\) and \(b\) and \(c\) are real numbers for which \(b^2 \lt 4c\text{,}\) then the operator
    \begin{equation*} T^2 + bT + cI \end{equation*}
    is invertible.

    Proof.

    We only need to show that \(\nul(T)=\{\zerovec\}\text{.}\) Therefore, we suppose that \(\vvec\) is a nonzero vector and consider
    \begin{equation*} \begin{aligned} \inner{(T^2+bT+cI)\vvec}{\vvec} \amp = \inner{T^2\vvec}{\vvec}+ \inner{bT\vvec}{\vvec}+c\inner{\vvec}{\vvec} \\ \amp = \inner{T\vvec}{T\vvec}+ b\inner{T\vvec}{\vvec}+c\inner{\vvec}{\vvec}\\ \amp = \len{T\vvec}^2 + b\inner{T\vvec}{\vvec}+c\len{\vvec}^2 \\ \amp \geq \len{T\vvec}^2 - b\len{T\vvec}\len{\vvec} + c\len{\vvec} \\ \amp \geq \left(\len{T\vvec} - \frac b2\len{\vvec}\right)^2 + \left(c-\frac{b^2}{4}\right)\len{\vvec}^2 \\ \amp \gt 0 \\ \end{aligned}\text{.} \end{equation*}
    This shows that \((T^2+bT+cI)\vvec\) is also nonzero so the operator is invertible.
    If \(\field=\complex\) and \(T\) is an operator on \(V\text{,}\) we know from the Theorem 2.1.4 that the minimal polynomial of \(T\) has the form
    \begin{equation*} p(x)=(x-\lambda_1)(x-\lambda_2)\ldots(x-\lambda_m) \end{equation*}
    where each \(\lambda_j\in\complex\text{.}\)
    Now suppose that \(\field=\real\) and \(T\) is a self-adjoint operator on \(V\text{.}\) We can make a similar conclusion.

    Proposition 2.3.4.

    If \(V\) is a real inner produce space and \(T\) is a self-adjoint operator on \(T\text{,}\) then the minimal polynomial of \(T\) has the form
    \begin{equation*} p(x)=(x-\lambda_1)(x-\lambda_2)\ldots(x-\lambda_m) \end{equation*}
    where \(\lambda_j\in\real\text{.}\)

    Proof.

    We know that the minimal polynomial has the form
    \begin{equation*} p(x)=(x-\lambda_1)\ldots(x-\lambda_s)(x^2+b_1x+c_1)\ldots (x^2+b_tx+c_t) \end{equation*}
    where \(b_i^2 \lt 4c-i\) for each \(i\text{.}\) However, since \(p(T)=0\text{,}\) we know that
    \begin{equation*} p(T)=(T-\lambda_1)\ldots(T-\lambda_s)(T^2+b_1T+c_1I)\ldots (T^2+b_tT+c_tI)\text{.} \end{equation*}
    If \(t\gt 0\text{,}\) then \(T^2+b_1T+c_1I\) is invertible by Lemma 2.3.3. If we multiply \(p(T)\) by its inverse, we obtain another polynomial \(q\) of smaller degree for which \(q(T)=0\text{.}\)
    Since the minimal polynomial \(p\) is the polynomial having the smallest degree among all polynomials for which \(p(T)=0\text{,}\) we conclude that \(t=0\) and therefore
    \begin{equation*} p(x)=(x-\lambda_1)\ldots(x-\lambda_m)\text{.} \end{equation*}

    Theorem 2.3.5. The Spectral Theorem.

    If \(T\) is a self-adjoint operator on a vector space \(V\text{,}\) then there is an orthonormal basis such that the matrix associated to \(T\) is diagonal.

    Proof.

    By Theorem 2.2.3 and Proposition 2.3.4, we know that there is a basis \(\vvec_1,\vvec_2,\ldots,\vvec_n\) of \(V\) for which the matrix associated to \(T\) is upper triangular.
    If we let \(V_j=\laspan{\vvec_1,\ldots,\vvec_j}\text{,}\) we know that \(V_j\) is invariant under \(T\text{.}\) We now apply the Gram-Schmidt algorithm to this basis to produce an orthonormal basis \(\uvec_1,\uvec_2,\ldots,\uvec_n\) for \(V\text{.}\) By the Gram-Schmidt algorithm, we know that
    \begin{equation*} V_j=\laspan{\vvec_1,\ldots,\vvec_j}= \laspan{\uvec_1,\ldots,\uvec_j}\text{,} \end{equation*}
    which says that \(\uvec_1,\ldots,\uvec_n\) is an orthonormal basis for \(V\) for which the matrix associated to \(T\) is upper triangular.
    However, if \(A\) is this matrix, we know that \(A^*\) is the matrix associated to \(T^*\text{.}\) Since \(T=T^*\text{,}\) we have \(A=A^*\text{,}\) which says that \(A\) is diagonal
    Notice that we have given a number of proofs that all the eigenvalues of \(T\) are real. This also shows that the eigenspaces associated with different eigenvalues are orthogonal to one another.
