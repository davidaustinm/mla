Proposition 2.2.1.

If \(\nul(T^m)=\nul(T^{m+1})\text{,}\) then \(\nul(T^{m+k}) = \nul(T^m)\) for any \(k\gt 0\text{.}\)

Proof.

Suppose that \(\nul(T^n) = \nul(T^{n+1})\) for some \(n\) and that \(\vvec\) is a vector in \(\nul(T^{n+2})\text{.}\) It follows that
\begin{equation*} T^{n+2}\vvec = T^{n+1}(T\vvec) = 0\text{,} \end{equation*}
which means that \(T\vvec\) is in \(\nul(T^{n+1})\text{.}\) Because \(\nul(T^{n+1}) = \nul(T^n)\text{,}\) it follows that \(T\vvec\) is in \(\nul(T^n)\text{,}\) which says that
\begin{equation*} T^{n}(T\vvec) = T^{n+1}\vvec = 0\text{.} \end{equation*}
We then see that
\begin{equation*} \nul(T^m)=\nul(T^{m+1})=\nul(T^{m+2}) = \nul(T^{m+3}) = \ldots\text{.} \end{equation*}
