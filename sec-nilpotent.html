    Section 2.4 Nilpotent operators

    Eigenvectors of an operator \(T\) are found through the equation \((T-\lambda I)\vvec=\zerovec\text{.}\) Now that we have developed some conditions under which operators are diagonalizable, we would like to understand what happens when operators are not diagonalizable. To this end, we will generalize the eigenvector condition by considering
    \begin{equation*} (T-\lambda I)^k\vvec=\zerovec\text{.} \end{equation*}
    To get started, however, we will first consider a related class of operators.

    Subsection 2.4.1 Null spaces of powers

    Suppose that \(T\) is an operator. If \(T^k\vvec=\zerovec\text{,}\) then it also happens that
    \begin{equation*} T^{k+1}\vvec=T(T^k\vvec)=\zerovec\text{.} \end{equation*}
    This means that \(\nul(T^k)\subset\nul(T^{k+1})\text{,}\) and we therefore have
    \begin{equation*} \nul(T)\subset\nul(T^2)\subset\nul(T^3)\subset\ldots\text{.} \end{equation*}
    The next propositions say that this process stabilizes so that the inclusions eventually become equalities. First we show that when we reach an equality, then all the following are inclusions.

    Proposition 2.4.1.

    If \(\nul(T^m)=\nul(T^{m+1})\text{,}\) then \(\nul(T^{m+k}) = \nul(T^m)\) for any \(k\gt 0\text{.}\)

    Proof.

    We will proof this fact using induction on \(k\text{.}\) Notice that the result is true for \(k=1\) by the assumption.
    We now assume that \(\nul(T^{m+j})=\nul(T^m)\) for every \(j\lt k\) and that we have a vector \(\vvec\in\nul(T^{m+k})\text{.}\) We need to show that \(\vvec\in\nul(T^m)\text{.}\)
    We know that
    \begin{equation*} T^{m+k}\vvec = T^{m+1}(T^{k-1}\vvec) = \zerovec\text{.} \end{equation*}
    Therefore, \(T^{k-1}\vvec\) is in \(\nul(T^{m+1}) = \nul(T^m)\text{,}\) which means that
    \begin{equation*} T^{m+k-1}\vvec=T^m(T^{k-1}\vvec)=\zerovec\text{.} \end{equation*}
    This means that \(\vvec\in\nul(T^{m+k-1}) = \nul(T^m)\) by induction.
    The next result says that this process will always stablize by the time we reach the dimension of \(V\text{.}\)

    Proposition 2.4.2.

    For any operator \(T\) on a vector space \(V\) of dimension \(n\text{,}\)
    \begin{equation*} \nul(T^n)=\nul(T^{n+1})\text{.} \end{equation*}

    Proof.

    If \(\nul(T)=\{\zerovec\}\text{,}\) then \(T\) is invertible as is every power of \(T\text{.}\) Therefore \(\nul(T^k)=\{\zerovec\}\) for every power, including \(n\) and \(n+1\text{.}\)
    Now suppose that \(\nul(T)\) has positive dimension so that \(\dim\nul(T) \geq 1\) and that the null spaces continue to grow
    \begin{equation*} \nul(T)\subsetneq\nul(T^2)\ldots\subsetneq\nul(T^m)\text{.} \end{equation*}
    In this case,
    \begin{equation*} n=\dim(V)\geq\dim\nul(T^m)\geq m\text{.} \end{equation*}
    This shows that the null spaces cannot grow beyond \(T^n\) so we have \(\nul(T^n)=\nul(T^{n+1})\text{.}\)

    Subsection 2.4.2 Nilpotent operators

    We will say that an operator is nilpotent if some power of the operator is zero.

    Definition 2.4.3. Nilpotent operator.

    An operator \(T\) is nilpotent if some power \(T^m=0\text{.}\)
    Notice that the operator \(T=0\) is nilpotent but we will often consider nonzero nilpotent operators.

    Example 2.4.4.

    Consider the matrix \(A = \begin{bmatrix} 0 \amp 1 \\ 0 \amp 0 \\ \end{bmatrix}\) and notice that \(A^2=0\text{.}\) An operator \(T\) whose associated matrix is \(A\) with respect to some basis is nilpotent since \(T^2=0\text{.}\)
    Suppose \(T\) is nilpotent and that \(m\) is the smallest power for which \(T^m=0\text{.}\) This means that \(p(T)=0\) if \(p(x)=x^m\) and hence \(p\) is the minimal polynomial of \(T\text{.}\) We could view
    \begin{equation*} p(x) = x^m = (x-0)^m\text{,} \end{equation*}
    which says that there is a basis for which the matrix associated to \(T\) is upper triangular
    \begin{equation*} \begin{bmatrix} 0 \amp * \amp * \amp \ldots \amp * \\ 0 \amp 0 \amp * \amp \ldots \amp * \\ \vdots \amp \vdots \amp \vdots \amp \ddots \amp * \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \\ \end{bmatrix}\text{.} \end{equation*}
    In fact, we will see that there is a basis so that the matrix associated to a nilpotent operator has an especially nice form.

    Definition 2.4.5.

    A nilpotent block matrix is a matrix having the form
    \begin{equation*} \begin{bmatrix} 0 \amp 1 \amp 0 \amp \ldots \amp 0 \\ 0 \amp 0 \amp 1 \amp \ldots \amp 0 \\ \vdots \amp \vdots \amp \vdots \amp \ddots \amp 1 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \\ \end{bmatrix}\text{.} \end{equation*}
    This is, all the entries are zero except for the entries directly above the diagonal, which are 1.

    Example 2.4.6.

    The following matrix consists of three nilpotent blocks on the diagonal, a \(2\times2\) block, a \(3\times3\) block, and a \(1\times1\) block.
    \begin{equation*} \begin{bmatrix} 0 \amp 1 \amp 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \\ \end{bmatrix}\text{.} \end{equation*}
    In fact, every nilpotent operator has basis whose associated matrix consists of a set of nilpotent blocks on the diagonal.

    Proposition 2.4.7.

    If \(T\) is a nilpotent operator on \(V\text{,}\) then there is a basis for \(V\) such that the matrix associated to \(T\) has the form
    \begin{equation*} \begin{bmatrix} A_1 \amp 0 \amp \ldots \amp 0 \\ 0 \amp A_2 \amp \ldots \amp 0 \\ 0 \amp \vdots \amp \ddots \amp 0 \\ 0 \amp 0 \amp 0 \amp A_k \\ \end{bmatrix} \end{equation*}
    where each \(A_j\) is a nilpotent block.

    Proof.

    We will prove this result using induction on \(n=\dim V\text{.}\) To begin, if \(n=\dim V = 1\text{,}\) then \(T=0\) so the matrix associated to \(T\) in any basis is \(\begin{bmatrix}0\end{bmatrix}\text{,}\) a \(1\times1\) nilpotent block.
    Now suppose that the result is true for every dimension less than \(n\) and suppose that \(m\) is the smallest integer for which \(T^m=0\) but \(T^{m-1}\neq 0\text{.}\) If \(m=1\text{,}\) then \(T=0\) and the matrix associated to \(T\) is the zero matrix, which is a collection of \(1\times 1\) nilpotent blocks
    Now suppose that \(m\gt 1\) and notice that our assumption implies that \(\nul(T^{m-1})\subsetneq\nul(T^m)\text{.}\) Therefore, there is a vector \(\vvec_m\) such that \(T^m\vvec_m=\zerovec\) but \(T^{m-1}\vvec_m \neq \zerovec\text{.}\)
    We define \(\vvec_j = T^{m-j}\vvec_m\) and note that \(T^j\vvec_j=\zerovec\) but \(T^{j-1}\vvec_j\neq \zerovec\text{.}\) Notice that the vectors \(\vvec_j\) form a linearly independent set. To see why, suppose we can find coefficients \(a_j\) such that
    \begin{equation*} a_1\vvec_1 + a_2\vvec_2 + \ldots + a_m\vvec_m = \zerovec.\text{.} \end{equation*}
    If we apply \(T^{m-1}\) to this vector, we have
    \begin{equation*} a_m\vvec_1 = \zerovec \end{equation*}
    which says that \(a_m=0\text{.}\) Continuing in this way, shows that each \(a_j=0\text{,}\) which shows that the vectors are linearly independent. We define the \(m\)-dimensional subspace \(W\) to be the span of these vectors.
    Notice that \(W\) is invariant under \(T\) and that the matrix associated to \(T|_W\) in this basis is an \(m\times m\) nilpotent block. If \(W=V\text{,}\) then the matrix associated to \(T\) in this basis is a single \(n\times n\) nilpotent block, and the conclusion of the theorem is true.
    So now suppose that \(\dim W \lt \dim V = n\text{.}\) We also know that \(\dim W = m \gt 1\) so we have
    \begin{equation*} 1 \lt \dim W \lt \dim V = n\text{.} \end{equation*}
    Notice that \(W\) is invariant under \(T\) and that the matrix associated to \(T|_W\) in the basis \(\vvec_1,\vvec_2,\ldots,\vvec_m\) is a \(m\times m\) nilpotent block. To finish the proof, we need to find a complement to \(W\) inside \(V\) that is invariant under \(T\text{.}\)
    To do so, we choose a linear functional \(\phi\in V'\) satisfying \(\phi(\vvec_1)= 1\) and \(\phi(\vvec_j)=0\) for \(j \gt 1\text{.}\) We use \(\phi\) to define the linear transformation \(\Phi:V\to\real^m\) by
    \begin{equation*} \Phi(\vvec) = (\phi(\vvec), \phi(T\vvec),\ldots,\phi(T^{m-1}\vvec))\text{.} \end{equation*}
    For a vector \(\wvec\in W\text{,}\) we can write
    \begin{equation*} \wvec = a_1\vvec_1 + a_2\vvec_2 + \ldots + a_m\vvec_m\text{.} \end{equation*}
    Notice that
    \begin{equation*} \begin{aligned} \wvec \amp = a_1\vvec_1 + a_2\vvec_2 + \ldots + a_m\vvec_m \\ T\wvec \amp = a_2\vvec_1 + a_3\vvec_2 + \ldots + a_m\vvec_{m-1} \\ T^2\wvec \amp = a_3\vvec_1 + a_4\vvec_2 + \ldots + a_m\vvec_{m-2} \\ \end{aligned} \end{equation*}
    so that
    \begin{equation*} \Phi(\wvec) = (a_1,a_2,\ldots,a_m)\text{.} \end{equation*}
    From this, we see that \(\Phi\) is surjective.
    We define the subspace \(U=\nul(\Phi)\) and note that \(U\cap W = \{\zerovec\}\) since \(\Phi(\wvec) \neq \zerovec\) if \(\wvec\) is a nonzero vector in \(W\text{.}\) Moreover, \(\dim U + \dim W = \dim V\) since \(\Phi\) is surjective. Therefore \(V = U\oplus W\text{.}\)
    All that remains is to show that \(U\) is invariant under \(T\text{.}\) Suppose that
    \begin{equation*} \Phi(\vvec) = (\phi(\vvec),\phi(T\vvec),\ldots,\phi(T^{m-1}\vvec)) = (a_1,a_2,\ldots,a_m)\text{.} \end{equation*}
    Then it follows that
    \begin{equation*} \Phi(T\vvec) = (\phi(T\vvec),\phi(T^2\vvec),\ldots,\phi(T^{m}\vvec)) = (a_2,a_3,\ldots,a_m,0)\text{.} \end{equation*}
    Therefore, if \(\Phi(\uvec) = \zerovec\text{,}\) then \(\Phi(T\uvec)=\zerovec\text{,}\) which says that \(U\) is invariant under \(T\text{.}\) Moreover, \(T|_U\) is still nilpotent so we know that the there is a basis for \(U\) for which the matrix of \(T|_U\) consists of nilpotent blocks. Combining this basis with the basis of \(W\) we have constructed gives a basis of \(V\) for which the matrix associated to \(T\) consists of nilpotent blocks.
